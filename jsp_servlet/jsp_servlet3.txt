Servlet:

A Servlet is a Java program that runs on a server and is used to create dynamic web content.
It acts as a middle layer between a web browser (client) and the database or business logic (server side).

Why Use Servlets?
 Platform independent (Java-based).
 Can use JDBC for database access.
 Foundation for JSP, Spring MVC, etc.


Jsp:
JSP stands for Java Server Pages.
 It is used to create dynamic web pages using Java + HTML.
 Works on top of Servlet technology — every JSP is internally converted into a Servlet.


Why Use JSP?
 Easier than Servlets
 Separation of Design & Logic
 Automatic Servlet Conversion
 Built-in Objects
 Supports Reusability
 Integration with JavaBeans & MVC
 support tag library(jstl)
