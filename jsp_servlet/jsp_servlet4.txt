✅ 2. How many digits are in a port number?
✔ Port numbers are 16-bit → maximum 5 digits

Min: 0

Max: 65535 → 5 digits

Common examples:

8080 → 4 digits

3306 → 4 digits

443 → 3 digits

65535 → 5 digits (maximum possible)



✅ Q4. Why port number is not 6 digits?

Because a port number is stored in 16 bits.

Maximum value of 16-bit = 2^16 – 1 = 65535

So port cannot go beyond 5 digits.

If port was 6 digits → it would require more than 16 bits → not allowed by networking standards.



✅ 3. Unicode range in Java

Java uses UTF-16, so:

✔ Unicode range = U+0000 to U+10FFFF
✔ Java char range:

Since char is 16-bit:

Min: '\u0000' (0)

Max: '\uFFFF' (65,535)
