firstly create docker compose file (so no need to write lengthy command)

docker-compose.yml or yaml  (create in same folder where also docker file present)

PyCharm -python package --> Docker Compose (package) download

Docker compose package -It is used for jab ham docker compose file bnane mai like when you type it will give you suggestion ,auto format krne mai hep krta hai

-----------------------------------------------------------------------------------------------------------------------------------------

docker-compose.yml

--------------------
services:  //services kind of container ,images (konsi images use krna hai)
  mysqldb:    //suppose mai MySQL ke liye bna rhahu ,mai name deta hu mysqldb (yeha pe kuch bi apne according name de sakte ho )
  images: 'Mysql:latest' //let assume aap sql container related  run krna hai 

jab bhi docker compose file run kru that time MySQL latest image use krkr run kre
  
d- detached abhi jrurut nhi hai
  environment: 
      - MYSQL_ROOT_PASSWORD="root"  //phle vala env var
      - MYSQL_DATABASE="userinfo"  //2nd vala en var
   
   container_name: "mysqldb"   //apne run krte wakt diya tha container name ,nhi diya tho bhi chlega , vo automatic radom default nam use                     kta hai
