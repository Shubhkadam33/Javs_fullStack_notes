constructor injection ->
constructor injection is a type of dependency injection (DI) where the spring container injects dependencies through a class constuctor

when spring create the object(bean) ,it passes the required dependencies as constructor parameter

Student.java

private int id;
private String name;

public Student() {
    System.out.println("Default Constructor");
}

// single parameter constructor
public Student(int id) {
    this.id = id;
}

public Student(String name) {   //if you want to pass only one argument then you can pass
    this.name = name;
}

public Student(int id, String name) {     //if you want pass two argument then you can pass
    this.id = id;
    this.name = name;
}
