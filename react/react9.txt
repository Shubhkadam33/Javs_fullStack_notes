solution: <> </>  empty opening tag and closing tag

function App(){
   return (
       
       <>
          <Title/>
          <Title/>
          <Title/>         
      </>
   );
}



<> </> - iska simple yek hi use hai which is yeh mutiple element/component/children ko combine krta hai in a single entity

now inspect:

<!DOCTYPE html>
<html lang="en">
