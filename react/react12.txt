there is two way to styling component:

              style.css
              /    |   |__________productlist.jsx
        app.jsx  product.jsx

it is correct but not standard way to write

>each component have separate css file ,we component wise divide css file

App.jsx -> App.css
product.jsx ->product.css
productlist.jsx -> productlist.css

notice: same component file name as css file name,it is not mandatory but standard way so every one understand 

eg: component filename, function name ,css filename =same name so easy to understand

-------------------------------------------------------------------------------------------

product.jsx:

function product(){
    return (
       <div className="product">
           <h3> product Title</h3>
           <h5> product Description</h5>
    )

standard way:  <div className="product">  generally outer/root element ko className dete hai kisi bhi component ke under className same hona chhiye as the component itself

 but you can write different name also  eg <div className="box1">
