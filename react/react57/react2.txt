handling Non-click Events

Nonclick event ke under bhot sare events aate hai ,onclick event generally button par dekhte hai

NDN documentation -> Event reference --> you can see click events and non click events(you can try events which you want to)

eg :dbclick event(double click) ,mouseover event

function handleClick(){
     console.log("Hello");
}

function handleMouseOver(){
      console.log("bye");
}

function handleDbClick(){
   console.log("you double clicked");
}

export default function Button(){
         return(
           <div>
              <button onclick={handleClick}>click me!</button>

              <p onClick={handleMouseOver}>Lorem ipsum, dolor sit amet consectetur adipisicing elit. 
                 Reprehenderit sequi accusamus similique repudiandae esse atque consequuntur unde eligendi asperiores ipsa, 
                 maxime alias ratione impedit nihil magnam aliquam nobis velit quod!
              </p>
                
               <button onDoubleClick={handleDbclick}>on double click!</button>
           </div>
         );
}
